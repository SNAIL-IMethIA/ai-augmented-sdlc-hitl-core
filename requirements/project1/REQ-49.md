# Searchable indicator glossary

**ID:** REQ-49

**Type:** Functional

**Originator:** Stakeholder / Client Interview

**Description:**
The system shall provide a searchable glossary of all supported indicators and trading concepts, accessible from within the application at any time.

**Rationale:**
A glossary ensures users can look up any term they encounter in strategies, metrics, or documentation without leaving the application, supporting both learning and professional review.

**Fit Criterion:**

- The glossary contains an entry for every built-in indicator included in the system.
- The glossary can be searched by keyword; matching results are returned within 1 second.
- The glossary is accessible from any screen within the application.

**Customer Satisfaction (0–5):**

- 3: Valuable reference tool that improves user competency over time.

**Customer Dissatisfaction (0–5):**

- 2: Users can look up terms externally, but this disrupts the in-application workflow.
