# Contextual hover tooltips

**ID:** REQ-47

**Type:** Functional

**Originator:** Stakeholder / Client Interview

**Description:**
The system shall display contextual hints and tooltips on all interactive user interface elements when hovered.

**Rationale:**
Hover hints provide just-in-time guidance without cluttering the interface, helping users discover functionality and understand controls without leaving the current context.

**Fit Criterion:**

- Every interactive UI element has an associated tooltip that appears within 500 ms of cursor hover.
- Each tooltip contains a plain-language description of the element's purpose or action.
- Tooltips are present across both Basic and Advanced modes.

**Customer Satisfaction (0–5):**

- 3: Improves discoverability and reduces reliance on external documentation.

**Customer Dissatisfaction (0–5):**

- 2: Users can consult external documentation, but in-context guidance is faster and less disruptive.
