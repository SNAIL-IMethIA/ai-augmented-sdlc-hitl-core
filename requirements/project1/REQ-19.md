# Multi-select pipeline elements

**ID:** REQ-19

**Type:** Functional

**Originator:** Stakeholder / Client Interview

**Description:**
The system shall allow a user to select multiple pipeline elements simultaneously via marquee (rubber-band) selection or modifier-key click.

**Rationale:**
Multi-selection enables bulk operations (move, copy, delete) and significantly improves editing efficiency for complex pipelines.

**Fit Criterion:**

- A user can select at least two elements in a single interaction using either marquee selection or modifier-key click.
- All selected elements are visually distinguished from unselected elements.

**Customer Satisfaction (0–5):**

- 3: Convenience feature that accelerates editing for complex strategies.

**Customer Dissatisfaction (0–5):**

- 2: Users can select and operate on elements individually, but with higher effort.
