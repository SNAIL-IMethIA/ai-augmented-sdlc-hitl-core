# No-code drag-and-drop builder

**ID:** REQ-16

**Type:** Functional

**Originator:** Stakeholder / Client Interview

**Description:**
The system shall provide a no-code, drag-and-drop interface for assembling strategy building blocks.

**Rationale:**
Eliminating code entry requirements makes the tool accessible to non-programmers, expanding the potential user base.

**Fit Criterion:**

- A complete strategy pipeline can be built without entering any code.
- All block placement and connection interactions can be performed using only pointer input (mouse or touch).

**Customer Satisfaction (0–5):**

- 5: No-code interaction is the primary product differentiator.

**Customer Dissatisfaction (0–5):**

- 5: Without a no-code interface, the tool is inaccessible to non-programmer users.
